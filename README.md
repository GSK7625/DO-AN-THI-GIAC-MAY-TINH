# TGMT - Crowd Counting and Person Detection

This project focuses on crowd counting and person detection using CSRNet and YOLOv8 on the ShanghaiTech dataset.

## Project Structure
- `data/`: Dataset configuration and scripts.
- `models/`: Implementations of CSRNet and YOLOv8 training logic.
- `notebooks/`: Data exploration and preprocessing.
- `requirements.txt`: Project dependencies.

## Setup
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Prepare the dataset using `notebooks/preprocessing.ipynb`.
3. Train models in `models/`.
